# 🧠 Diabetes Prediction App

A simple ML-powered web app to predict diabetes risk based on basic health inputs.

Built using:

- Python
- Scikit-learn
- Streamlit

---

## 🚀 Features

- User-friendly health input form
- Real-time prediction
- Clean dashboard with results
- Basic health tips

---

## Tech Used

- Python
- Scikit-learn
- Streamlit

## How to Run

1. Create virtual environment

python -m venv diabetes_env
diabetes_env\Scripts\activate

2. Install dependencies

pip install -r requirements.txt

3. Run the app
   streamlit run app.py

## About

- Takes user health inputs
- Predicts diabetes risk using a trained model
- Shows result with a simple interface
